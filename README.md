# BusMapLearning
창의자율과제 버스맵러닝
-------------------------------

## Overview

### 목적
 - 부산 시내 대중교통 수단(버스, 지하철)의 정류장별 혼잡도를 분석
 - 정류소별 승하차 인원을 학습하여 정류소의 위치 및 배차간격 등을 변경하였을 때에의 혼잡도를 예측하는 모델 제작
